How to use Budget Studio

Set up your workspace once, then keep it current as money moves.

Recommended setup order

Accounts → Budget → Transactions → Overview

Complete these steps in order so every transaction has the right account and category.

Start setup
01

Add your accounts

Open Budget and Account Settings. Add each bank account, credit card, cash wallet, or investment account you want to track.

Examples: Main Bank, Savings, Cash, Visa Card

02

Create categories and budgets

Add fixed commitments under Bills and flexible spending under Monthly Expenses. Enter the amount you plan to spend each month.

Bills: Rent, Insurance · Monthly Expenses: Food, Transport

03

Set up income types

Add the income subcategories you use, such as Salary, Bonus, Freelance, Dividend, or Interest.

04

Add and edit transactions

Open Add transaction, choose Income or Expense, complete the details, and save. Use the table below the form to view or edit existing transactions.

05

Review your overview

The Overview calculates income, expenses, cash flow, category spending, and budget balances. Use its date, category, and account filters to investigate your finances.

06

Keep your data accurate

Add transactions regularly and correct mistakes with Edit. Changes are saved securely to the currently signed-in account.

Quick help

Dropdown is empty
Create the relevant account or budget category in Settings first.
Dashboard has no values
Add transactions, then clear any active filters on Overview.
Need to correct an entry
Open Add transaction, find the entry in All transactions, and select Edit.
Using another Google account
Each signed-in account has its own separate financial data.
